# Network-Worm-Containment-with-Azure-NSGs
This repository contains supporting material for the project “Network Worm Containment Using Azure Network Security Groups.”

Contents
	•	Azure deployment details
	•	Commands used for baseline testing
	•	Metasploit auxiliary modules used for service visibility
	•	Network Security Group rule configuration
	•	Virtual network flow log samples

Environment
	•	Cloud platform: Microsoft Azure
	•	OS: Ubuntu Server
	•	Tools: Ping, Telnet, Metasploit Framework

Purpose
The project demonstrates how subnet-level Network Security Group rules can block internal lateral movement in a cloud network.

Notes
All testing was conducted in an isolated academic environment. No real malware was deployed.
